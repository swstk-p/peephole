# Peephole

A browser based biometric credential management system that utilizes face-recognition authentication and encrypted credential storage along with automated form fill for login.

## Overview

Peephole is a proof of concept credential management system developed as an academic project for BCA. It uses the extension mechanisms of Chrome browser in order to provide functionalities of face recognition for authentication and credential management as well as automated form fill ups for login geared towards simplifying the process of authentication on websites.

The project, in overview, consists of:
- React-based UI for the Chrome extension
- Node.js and Express backend
- MongoDB based credential storage
- Face-api.js for face detection and recognition

 Once authenticated, users can manage their stored credentials - indexed by websites - and are able to utilize automated form fill ups for simplified login.
